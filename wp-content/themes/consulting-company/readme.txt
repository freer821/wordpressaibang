== Theme Description ==
Consulting Company is a WordPress theme developed primarily for consulting companies but it can be used for any company which needs an impressive, mobile and SEO friendly website using a free WordPress theme. Consulting Company WordPress theme is easily customizable using Customizer settings of WordPress and extremely easy to setup with the help of free dummy data and step by step documentation.https://cipherthemes.com/documentation/consulting-company

== Installation ==
1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "consulting-company.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (consulting-company.zip) that you have downloaded.
 = Upload the entire folder (consulting-company) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/consulting-company/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "consulting-company" to activate the theme.

5. Complete all of the required inputs in the Appearance => Custimize option and click "Save & Publish" to make the changes LIVE.
